HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Abovethe head, "Opening Tag" tells us it's an HTML file
  • Also above the head, an "atribute" defines additional information about an element. Can tell us it's in English.
  • Semantic HTML element: with these, the name of the element describes its content and the role it plays on a webpage. When possible, it is important to use semantic HTML elements, because they make your HTML easier to read by other developers and the browser
  • Element: element refers to the entire element from opening tag to closing tag
  • Tag: tag refers only to what appears inside the angle brackets
